Ubuntu 系统的引导修复

  • 2015-05-26
  • | 标签:
  • 记录
  • Ubuntu
  • GRUB

比如说在一台 Ubuntu 的电脑上再安装了 Windows, 那么你就只能进入 Windows 了。
以下是修复方法：

用和原系统匹配的 Ubuntu 引导盘，引导进入 “Try Ubuntu”, 然后打开终端即可输入命令了

  1. 获取 root 权限

    1
    sudo -i

  2. 查看分区情况

    1
    fdisk -l

  3. 挂载需要修复引导的系统的 / 目录，可由上一步查看到需要挂载哪个目录，
    如若不确定，可尝试挂载以下，看看里面有什么内容再来确定

    1
    mount /dev/sdx1 /mnt

  4. 如果 /boot 分区是单独分区的，那么也需要挂载需

    1
    mount /dev/sdx2 /mnt/boot

  5. 挂载运行系统目录到需要修复引导的系统下

    1
    for i in /dev /dev/pts /proc /sys; do sudo mount -B $i /mnt$i; done

  6. 进入需要修复引导的系统

    1
    chroot /mnt

  7. 重新安装 GRUB 到 MBR, 并更新 GRUB

    1
2
    grub-install /dev/sdx
update-grub

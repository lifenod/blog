插件安装及使用方法

  1. 关闭服务器

  2. 将插件放到 plugins 目录
    再启动服务器，插件就正常运行了，也会在 plugins 目录生成相应的配置文件

  3. 如果需要修改插件的配置，那么再次将服务器关闭，修改完之后再次启动

    也可以不用重启服务器，将配置修改好之后，使用插件的 “Reload” 命令

AuthMe Reloaded

项目主页：https://github.com/Xephi/AuthMeReloaded

登录/注册插件，非常好用

SkinsRestorer

主页：https://www.spigotmc.org/resources/skinsrestorer.2124/

如果服务器为 Offline 模式，那么正版用户也无法显示皮肤，这款插件就是为了修复这一问题

依赖这款插件 ProtocolLib: https://www.spigotmc.org/resources/protocollib.1997/

所以要将以上两者都放入 plugins 目录

Simple Backup

项目主页：https://github.com/Exolius/SimpleBackup

一款非常简单好用的备份插件，用来备份世界地图

Minecraft 服务器 PaperSpigot 的使用

  • 2015-09-19
  • | 标签:
  • 记录
  • Minecraft
  • PaperSpigot
  • Game

编译

  1. 去 https://ci.destroystokyo.com/job/PaperSpigot-BuildTools/ 下载构建工具，下载到文件为 PaperTools.jar

  2. 新建一个目录，并将 PaperTools.jar 放到里面

  3. 自动编译

    java -jar PaperTools.jar

    等好一会之后，例如会在当前目录下生成 paperspigot-1.8.8.jar 文件，这就是我们要的最终文件了

运行

和 Minecraft 官方服务器用法差不多，paperspigot-1.8.8.jar 就相当于官方服务器的 minecraft_server.1.8.8.jar

Minecraft 官方服务器用法可查看我的一篇文章《架设 Minecraft 官方服务器》

比官方服务器多的那一部分

官方服务器主要配置文件就一个 server.properties
PaperSpigot 中的配置文件为 server.properties, bukkit.yml, spigot.yml, paper.yml, 可配置部分也多多了

以下只列举出了配置的很少一部分

bukkit.yml

详情见：http://wiki.bukkit.org/Bukkit.yml

# 一个区块中允许的生成生物数
# 刷怪箱不受其影响，是另一套机制
spawn-limits:
  monsters: 70
  animals: 15
  water-animals: 5
  ambient: 15

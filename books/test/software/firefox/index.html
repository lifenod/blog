# Firefox - 浏览器

下载：https://www.mozilla.org/en-US/firefox/all/

# 修改开始页面

打开新窗口居然默认打开 start.firefoxchina.cn，仅仅是因为我设置语言为中文吗，但是我下载浏览器是国际版的。什么鬼，一定要改掉。

  1. 地址栏输入 about:config
  2. 改掉 browser.startup.homepage 的值为 about:newtab, 这样打开新窗口的时候会显示 newtab 页，而不是默认的 start.firefoxchina.cn。

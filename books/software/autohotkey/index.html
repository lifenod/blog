AutoHotkey

2015-05-11

官网：http://ahkscript.org/
一款开源的 Windows 下的自动化软件，最初是提供快捷键功能的，后来才发展成一门完整的脚本语言
所以它的快捷键功能依旧很强大

可以从这里学习：http://ahkscript.org/docs/Tutorial.htm

如果使用 Sublime Text 编辑器，那么可能需要 SublimeAutoHotkey 工具：https://github.com/ahkscript/SublimeAutoHotkey

热键

游戏组合键

以「魔能」游戏中的「加速术」为例，必需顺序按下 a, s, f, space 键可释放该魔法
我们写个脚本，使之按下 1 键即可释放该魔法

; 必需，在魔能这个游戏中，按键必需按下一定的时间  
; 两次按键间隔 5 ms, 一次按键按下 65 ms 后松开
SetKeyDelay , 5, 65
; 加速术
1::Send asf{space}

重映射按键

当你习惯了 Mac 电脑下的快捷键，例如 ⌘ + c 为复制，⌘ + v 为粘贴， 到 Windows 下的时候会非常不习惯（Windows 下 ⌃ + c 为复制，⌃ + v 为粘贴
为了将 Mac 下的习惯带到 Windows 下，我们可以在 Windows 下交换 ⌘ 和 ⌃ 按键

; 将左 Control 键映射成左 Windows 键，左 Windows 键键映射成左 Control 键
LCtrl::LWin
LWin::LCtrl

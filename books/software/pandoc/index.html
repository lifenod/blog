Pandoc

2015-08-24

可实现不同标记语言间的格式转换，堪称该领域中的「瑞士军刀」
——维基百科

官网：http://pandoc.org/

Mac 下安装

  1. 使用 Homebrew 工具安装

    brew install pandoc

  2. 如果要转换 PDF 文档，那么需要 LaTeX 支持，可安装 MacTeX, 如果觉得太大了，也可安装 BasicTeX

  3. 如果安装了 BasicTeX, 并且在使用中遇到了字体问题，请运行如下命令

    tlmgr install collection-fontsrecommended

用法

  • 将 Markdown 转换为 PDF

    pandoc resume.md -o resume.pdf

  • 将 Markdown 转换为 Word

    pandoc resume.md -o resume.docx -c css/resume.css

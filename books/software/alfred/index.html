Alfred

2015-09-02 | 更新：2016-05-18

官网：https://www.alfredapp.com/

Max 下的 Spotlight 搜索功能也是非常强大
我觉得搜索功能比 Alfred 还有强大，几乎能搜索所有的东西，比如说联系人，Evernote 的笔记等。 而且搜索显示页面也挺好看的，还能够实时显示文件内容等

但是 Alfred 也有其优点，和 Spotlight 的搜索功能相比，Alfred 能够指定搜索的类型，比如文件，文件夹，应用，都能分开。 Spotlight 你只要告诉它你要搜索什么，它全部搜索出来了，有时候我们并不希望都被搜索到，那样可选条目就太多了
还有 Alfred 对于搜索出来的条目支持快捷键打开，比如 ⌘ + 1, ⌘ + 2 等， 而 Spotlight 则不能
还有 Alfred 的 Workflows 也非常优秀，可以完成一些小任务，比如切换 Apple ID, 查询电影等

两个工具都很好用，如果有个工具能够集合两者的优点，那就更好了

一些自己常用的功能

  • Clipboard
    它会把你复制过的内容记下来，通过快捷键调出这些内容。 当需要一次性复制很多的时候很有用，还有想再粘贴以前的内容也很有用。

  • Snippets
    功能：比如你输入 ",mai" 会自动扩展成 "blog@mwum.com"
    更新了 Alfred 3 之后，它支持直接扩展了，不再需要通过快捷键了。 我发现它比 aText 还要好用，aText 有时候在 Vim 下会失效（可能是我不会设置）， 但是它可以通过开启 "Expand snippets mid-string"，设置之后感觉任何时候都能扩展了。 而且也支持中文输入法，是时候该抛弃 aText 了 :)

  • Workflows

Tip

  • 使其搜索到用 HomeBrew 安装的应用

    Alfred Preferences → Features → Default Results → Search Scope → 点击加号 → ⇧ + ⌘ + g → 输入：/usr/local/Cellar → 点击 open

  • 使其搜索到用 HomeBrew Cask 安装的应用
    HomeBrew Cask 说从 Alfred v2.6 起已经被支持了，但是我现在已经 v2.7.2 了，还是不行， 或许是我从古老的版本升上来的缘故
    解决办法和 HomeBrew 一样

    Alfred Preferences → Features → Default Results → Search Scope → 点击加号 → ⇧ + ⌘ + g → 输入：/opt/homebrew-cask/Caskroom → 点击 open

NewFile

2016-02-02

项目主页：https://github.com/vitorgalvao/alfred-workflows#newfile

在当前目录下创建新文件
我们都知道 OS X 系统不能右键创建新文件，有时候会很不方便，所以这个插件很实用

用法：

  • 新建文件但不打开，nf filename1/filename2, / 用于分隔多个文件
  • 新建文件并打开，nfo filename

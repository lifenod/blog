1Writer

2016-02-13

官网：http://1writerapp.com/

iOS 下非常完美的文本编辑软件，漂亮，强大，简洁，简直完美
支持 URL Scheme, http://1writerapp.com/docs/urlscheme
可通过 JavaScript 编写 Action, http://1writerapp.com/docs/js, http://1writerapp.com/actiondir

值得一提的是它的插入图片功能也非常完美
首先你得开启 Dropbox 或 Evernote 等服务，然后当你在 Markdown 文件中插入图片的时候， 它会自动将图片上传到 Dropbox, 然后在文档中自动填入图片的共享地址， 这样无论在哪，只有你拥有 Markdown, 图片都可正常显示

用法

推荐看看这篇文章：https://www.macstories.net/ios/markdown-and-automation-experiments-with-1writer/

  • 设置默认使用 iCloud

    1. 文件夹页面 → 添加文件夹 → iCloud → 选择此文件夹
    2. 文件夹页面 → 设置 → 默认文件夹 → 选择 iCloud 文件夹 → 选择此文件夹

    这样你可以随时在 iCloud Drive 中查看纯文本文件

Disqus integration - 评论

让 GitBook 支持 Disqus 评论

2015-05-29

使用的工具是：plugin-disqus

安装方法一：

  1. 直接使用 npm 来安装插件，或者参数 -g 在全局安装

    npm install gitbook-plugin-disqus

  2. 在书籍主目录新建 book.json 文件，写入如下内容：

    {
    "plugins": ["disqus"],
    "pluginsConfig": {
        "disqus": {
            "shortName": "XXXXXXX"
        }
    }
}

安装方法二：

  1. 在 book.json 文件中写入如下内容，或者不加版本号

    {
    "plugins": ["disqus@0.0.1"],
    "pluginsConfig": {
        "disqus": {
            "shortName": "mwum"
        }
    }
}

  2. 在书籍主目录下执行如下命令：

    gitbook install

Homebrew

2015-06-07

官网：http://brew.sh/

Mac 上最流行的包管理器

安装

首先确保安装好 Command Line Tools for Xcode

然后使用如下命令来安装 Homebrew

ruby -e "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/install)"

添加环境变量，以 Zsh 为例

echo '' >> ~/.zshrc
echo '# brew' >> ~/.zshrc
echo 'export PATH=/usr/local/sbin:$PATH' >> ~/.zshrc

最后运行如下命令，查看 Homebrew 是否工作正常

brew doctor

使用

安装一个软件的流程

brew update
brew doctor
brew search foo
brew info FORMULA
brew install FORMULA
"Some configuration"

卸载一个软件的流程

brew list
brew uninstall FORMULA

更新一个软件的流程

brew update
brew outdated
brew upgrade FORMULA

以下是一些常用的命令，详情请看 man brew

安装软件

brew install FORMULA

卸载软件

brew uninstall FORMULA

更新 Homebrew 自身

brew update

检查系统中的 Homebrew 是否工作正常，一般在安装新软件之前执行一下

brew doctor

搜索软件

brew search foo

查看具体软件的信息，可先通过 brew search 命令查找到具体软件
安装软件之前执行一下，看看需要注意或者需要额外配置的地方

brew info FORMULA

列出已经安装的软件

brew list

查看哪些软件可更新

brew outdated

更新某软件

brew upgrade FORMULA

Homebrew 会保留旧的安装包，以便需要的时候可以回滚
若你想清理这些安装包，可运行如下命令

brew cleanup

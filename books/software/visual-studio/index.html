Visual Studio 2010

2015-00-00

官网：https://visualstudio.microsoft.com/

  • 安装了智能提示插件 VissualAssistX 之后，中文注释会出现红色波浪线，影响美观
    去除方法：VAssistX 菜单栏 → Visual Assist X Options → 展开Advanced → Underlines → 把 "underlines spelling errors in comments and strings using" 的勾去掉

  • 用空格代替 Tab, 强迫症需要整齐
    工具 → 选项 → 文本编辑 → 选择对应的语言 → 制表符

  • 显示空格和 Tab, 编辑器你想瞒着我什么！
    ⌃ + r, ⌃ + w 按此快捷键即可

  • 程序发布
    如果是动态编译的，那么当别人运行你的程序的时候，需要一些 DLL 文件才能运行。 有时候不能确定是哪些 DLL 文件的时候，可用 Dependency Walker: http://www.dependencywalker.com/ 来分析
    如果是静态编译的，那么别人运行你的程序的时候，只需要一个 EXE 文件

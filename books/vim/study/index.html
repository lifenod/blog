明无梦的 Vim 使用记录

学习

2015-09-05

Vim 上手非常简单

网上总能找到 Vim 陡峭的学习曲线，在我看来这种学习方式完全是自虐啊
相反我觉得学习 Vim 非常简单

下面我来说说上手步骤

  1. 下载好 GUI 版本的 Vim, 比如 MacVim
    然后你就可以像使用普通编辑器一样使用了，鼠标选中文字，使用系统快捷键复制粘贴， 鼠标定位。上下左右移动就是你键盘上的四个箭头，别告诉我你看不见。 只要你会使用电脑，我想这些你应该都会

    想输入文字了按下 "a" 或 "i" 键，输入完了按几次 "ESC" 键

    是的，用起来就是那么简单

  2. 需要去了解一些 Vim 的四个模式，然后尝试着使用 "k, j, h, l" 这四个上下左右快捷键

  3. 网上找一些配置文件，尝试配置一下 ~/.vimrc

  4. 各种插件的使用，各种快捷键的使用，并尝试更高端的配置

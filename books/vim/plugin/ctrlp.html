文件查找 (CtrlP)

文件查找 (CtrlP)

2014-00-00

主页：https://github.com/kien/ctrlp.vim

  • 支持模糊匹配
  • 可切换匹配模式，默认是 path, 其实很少需要切换

常用的功能

  • <c-p> 调用查找
  • <F5> 刷新，在当前目录下重建缓存
  • <c-y 在当前路径下创建新文件
  • <c-j>, <c-k> 在搜索列表中上下移动

配置

set wildignore+=*/tmp/*,*.so,*.swp,*.zip
let g:ctrlp_custom_ignore = {
  \ 'dir':  '\v[\/]\.(git|hg|svn)$',
  \ 'file': '\v\.(exe|so|dll|jpg|png|jpeg)$',
  \ }

nerdtree-git-plugin

nerdtree-git-plugin

2016-01-28

项目主页：https://github.com/Xuyuanp/nerdtree-git-plugin

在目录树中显示 git 状态，依赖于 The NERD Tree

可改变显示符号：

let g:NERDTreeIndicatorMapCustom = {
    \ "Modified"  : "✹",
    \ "Staged"    : "✚",
    \ "Untracked" : "✭",
    \ "Renamed"   : "➜",
    \ "Unmerged"  : "═",
    \ "Deleted"   : "✖",
    \ "Dirty"     : "✗",
    \ "Clean"     : "✔︎",
    \ "Unknown"   : "?"
    \ }

新系统设置

新系统设置

2016-12-12

macOS Sierra 10.12.1

  1. 重新安装 macOS：

    1. 如果开启了 FileVault，先将其关闭。
    2. 开机或重新启动后立即按住 ⌘ + R，看到 Apple 标志时松开这两个键，进入恢复功能
    3. 选择语言之后选择「重新安装 macOS」，按提示操作即可。

  2. 第一次开机设置：开启定位，登录 Apple ID 等，跳过设置都可以，如果都跳过了，之后也可以设置

  3. 系统偏好设置 → 触控板 → 跟踪速度 → 调到最快，默认的真的感觉好慢，手移动很大距离鼠标才移动一点点
    触控板 → 更多手势 → 勾选「应用 Expose」→ 当一个应用有很多页面的时候就有用了

  4. 系统偏好设置 → 键盘 → 键盘 → 将「重复前延迟」调到最短，要不然按住删除键，删除文字的速度太慢了
    键盘 → 键盘 → 点击右下角的「自定义控制条」可以将自己喜欢的内容加入 Touch Bar 中，比如我不用 Siri，就可以将这个位置换成其它的
    键盘 → 键盘 → 点击右下角的「修饰键」将 caps lock 键映射成 control 键，因为 caps lock 键真的几乎用不到，而且这有占了一个这么好的位置

  5. 省略。。。（将设置扫一遍，然后看着设置一下就好了）

更改计算机名

更改计算机名

2015-09-03 | 更新: 2016-12-18

一共有三个相关的名称：

  • ComputerName 共享的时候别人可以看到的名字，字面意思就是计算机名
  • LocalHostName 在本地网络中可以通过 LocalHostName.local 来访问这台电脑
  • HostName 网络中使用的名字，还有终端提示那里会显示这个，准确来说应该是计算机名加上域名

查看这些名称：

scutil --get ComputerName
scutil --get LocalHostName
scutil --get HostName

修改这些名称：

sudo scutil --set ComputerName NewName
sudo scutil --set LocalHostName NewName
sudo scutil --set HostName NewName

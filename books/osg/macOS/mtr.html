mtr - 网络诊断工具

mtr

2015-09-03

网络诊断工具，结合了 ping 和 traceroute 精华

安装

brew install mtr

使用

比如输入命令 mtr -r www.google.com
稍后会显示很多内容，具体含义请看：

  • Host: IP 地址或域名
  • Loss%: 每一跳的丢包率
  • Snt: 发送的数据包的数量
  • Last: 最近一次「数据包往返时间」
  • Avg : 「数据包往返时间」的平均值
  • Best: 最短的「数据包往返时间」
  • Wrst: 最长的「数据包往返时间」
  • StDev: 「数据包往返时间」的标准偏差，越大说明越不稳定

详细使用方法请看：《使用 MTR 诊断网络问题 | 每日一贴》

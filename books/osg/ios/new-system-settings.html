明无梦的操作系统使用记录

新系统设置

2015-08-15 | 更新：2015-10-16

iPad, iOS 9.0.2

用途：视频，游戏

  1. 通用 → 蓝牙 → 关闭
  2. 通用 → 关于本机 → 名称
  3. 通用 → Siri → 关闭
  4. 通用 → Spotlight 搜索 → Siri 建议 → 关闭
  5. 通用 → Handoff 与建议的应用 → 全部关闭
  6. 通用 → 辅助功能 → 减弱动态效果 → 打开
  7. 通用 → 后台应用刷新 → 后台应用刷新 → 关闭
  8. 声音 → 锁定声 → 关闭
  9. 声音 → 按键声 → 关闭
  10. 隐私 → 运动与健身 → 健身跟踪 → 关闭
  11. iCloud → 将不需要同步项目关闭
  12. App Store 与 iTunes Store → 全部关闭
  13. 邮件、通讯录、日历 → 获取新数据 → 推送 → 关闭
  14. 邮件、通讯录、日历 → iCloud → 手动
  15. 信息 → iMessage → 关闭
  16. FaceTime → FaceTime → 关闭
  17. 下拉菜单 → 编辑 → 移除不需要的 Widget

iPad, iOS 8.4

用途：看视频，玩游戏，刷社交

  1. 通用→关于本机→名称→自行修改
  2. 通用→辅助功能→减弱动画效果→打开
  3. 通用→后台应用程序刷新→后台应用程序刷新→关闭
  4. 通知→将不必要的通知关闭
  5. 显示与亮度→自动亮度调节→关闭
  6. 声音→锁定声→关闭
  7. 声音→按键音→关闭
  8. iCloud→将不需要同步的关闭
  9. iCloud→备份→iCloud 云备份→打开
  10. iTunes Store 与 App Store→自动下载的项目→全部关闭
  11. iTunes Store 与 App Store→建议的应用程序→全部关闭
  12. 信息→iMessage→关闭
  13. FaceTime→FaceTime→关闭
  14. Safari→搜索引擎→谷歌
  15. 下拉菜单→编辑→移除不需要的 Widget

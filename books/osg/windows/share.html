共享 (SMB/CIFS)

共享

2016-12-11

共享 Windows 中的某些文件夹，这样我就可以躺在床上用 iPad 观看 Windows 中的影片了。

开启共享：控制面板 → 网络和 Internet → 网络共享中心 → 左边的「更改高级共享设置」→ 选择启用文件共享的选择即可。
默认 C:\Users\Public 文件夹是共享的，你也可以自己添加其它文件夹。
只要右键这个文件夹 → 右键「属性」→ 点击「共享」→ 然后按自己的需要设置共享（可以设置权限等）。

用 SMB/CIFS 协议来连接就可以了，很多播放器都支持的。
顺便 macOS 也可以访问这些文件了。

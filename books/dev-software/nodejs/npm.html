明无梦的开发软件使用记录

NPM

Node.js 的官方包管理器，软件包可去如下网站查看：https://www.npmjs.com/

-g 选择表示作用于全局，否则作用于当前目录，例如：

在当前目录安装软件

npm install packageName

在全局安装软件

npm -g install packageName

列出软件

npm list

更新软件

npm update packageName

更新所有软件

npm update

删除软件

npm uninstall packageName

以上命令都可加上 -g 参数

jEnv - 管理 Java 环境变量

jEnv

2015-06-13

官网：http://www.jenv.be/

一个管理 Java 环境变量的工具

安装 (Mac and Zsh)

brew install jenv
echo 'export PATH="$HOME/.jenv/bin:$PATH"' >> ~/.zshrc
echo 'eval "$(jenv init -)"' >> ~/.zshrc
source ~/.zshrc

配置

添加 Apple 官方提供的 Java

jenv add /System/Library/Java/JavaVirtualMachines/1.6.0.jdk/Contents/Home

添加从 Oracle 下载的 Java

jenv add /Library/Java/JavaVirtualMachines/jdk1.7.0_79.jdk/Contents/Home

使用

列出所管理的 JDKs, 前面有 * 号的表示是当前使用的版本

jenv versions

配置全局生效

jenv global oracle64-1.7.0.79

配置当前目录生效

jenv local oracle64-1.7.0.79

配置当前 Shell 生效

jenv shell oracle64-1.7.0.79

查看 Java 版本是否配置正确

java -version

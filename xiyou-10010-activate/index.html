方法一（只支持 Windows）

http://tieba.baidu.com/p/3606416310,
使用此帖子中的插件
记得将 IP 改为 222.24.63.10

亲测其中的版本 01 可用

可用版本备份下载地址：https://www.dropbox.com/s/c56uadc0q6fjnv7/Freedrcom.zip?dl=0

方法二（支持各系统）

https://github.com/drcoms/drcom-generic,
其中的 p 版可用，《p 版简略使用和配置说明》

通过抓包获取到的信息为：

1
2
3
server = '222.24.63.10'
pppoe_flag = '\x0a'
keep_alive2_flag = '\xdc'

将以上内容替换 latest-pppoe.py 文件中的相应内容，然后运行此文件来激活（需要 Python 环境）

亲测路由器可使用（r6300v2 刷的 dd-wrt 系统）
简单说下步骤：

  1. 开启 SSH 和 JFFS

  2. 将 /jffs/opt 挂载到 /opt

  3. 用 ipkg 来安装 opkg

  4. 用 opkg 来安装 python

  5. 加入如下启动代码

    1
2
3
    mount --bind /jffs/opt /opt
sleep 15
/opt/bin/pppoe.sh

    需要修改 pppoe.sh 文件，很简单就不讲了

已修改好的 latest-pppoe.py 文件下载地址：https://www.dropbox.com/s/idtxn22vojyiunl/latest-pppoe.py?dl=0
